## Опис
У цій лабораторній роботі було розроблено Android-додаток для дослідження роботи **вбудованого датчика освітлення** мобільного пристрою.

Програма зчитує поточний рівень освітленості з датчика, обробляє отримані дані та відображає їх у зручному для користувача вигляді.

---

# Мета роботи
Ознайомитися з можливостями вбудованих датчиків мобільних пристроїв та дослідити способи їх використання для збору та обробки даних.

---

# Використаний датчик

У додатку використовується:

**Light Sensor (датчик освітлення)**

Він вимірює рівень освітленості навколишнього середовища в **люксах (lx)**.

---

# Функціональність додатка

Реалізований додаток має такі можливості:

- зчитування поточного значення освітленості
- відображення освітленості в **люксах**
- визначення рівня освітлення
- текстова **рекомендація користувачу**
- обчислення статистики:
  - мінімальне значення
  - максимальне значення
  - середнє значення
- візуалізація рівня освітлення за допомогою **ProgressBar**
- кнопка **Reset** для скидання статистики
- покращений інтерфейс користувача

---

# Логіка роботи програми

Після запуску програма отримує доступ до датчика освітлення через **SensorManager**.

При зміні освітлення викликається метод: onSensorChanged()

У цьому методі:

1. зчитується поточне значення освітлення
2. оновлюється текст на екрані
3. обчислюється статистика
4. оновлюється ProgressBar
5. визначається рівень освітлення
6. формується рекомендація для користувача

# Як запустити проєкт

1. Відкрити проєкт в **Android Studio**
2. Дочекатися синхронізації Gradle
3. Переконатися що `minSdk` ≥ **23**
4. Запустити застосунок на телефоні або емуляторі
5. Змінювати освітлення навколо пристрою

# Контрольні питання

## 1. Приклади вбудованих датчиків

- акселерометр — визначає прискорення
- гіроскоп — визначає кутову швидкість
- датчик освітлення — вимірює освітленість
- датчик наближення — визначає об'єкти біля телефону
- магнітометр — визначає напрямок
- барометр — вимірює атмосферний тиск

---

## 2. Особливості роботи з датчиками

- доступ до датчиків здійснюється через **SensorManager**
- датчик отримується методом **getDefaultSensor()**
- для отримання даних використовується **SensorEventListener**
- нові дані приходять у метод **onSensorChanged()**
- датчик реєструється у **onResume()**
- вимикається у **onPause()** для економії батареї
